Add a BKN301 button to your website and start selling with BKN301 E-commerce.

== Description ==

BKN301 plugin will allow you accept payments on your website using BKN301 innovate payment channel.
Your customers can use their various methods of payment to pay for your products and services.
You can easily install BKN301 E-commerce plugin on your website and accelerate sales.



== Installation ==

= Prerequisites =

- You must be using WooCommerce 3.0 or newer
- You must enable WordPress pretty permalinks in Settings > Permalinks.

= Installation Instructions =

- Upload the plugin files to WordPress.
- Activate the plugin through the Plugins screen in WordPress.
- Go to WooCommerce -> Settings -> Payments ->BKN301 (click Manage/Set up) and fill out required settings.
